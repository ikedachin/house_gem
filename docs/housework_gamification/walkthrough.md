# Housework Gamification MVP Walkthrough

## 概要
家事をゲーム化し、家族や同居人と楽しみながら分担するためのWebアプリケーションです。
MVP（Minimum Viable Product）として、以下の機能が実装されています。

- **アカウント管理**: ユーザー登録、ログイン
- **グループ管理**: 家グループの作成、招待コードによる参加
- **家事管理**: タスクの登録（難易度・ポイント設定）
- **実行記録**: 「やった！」ボタンでポイント獲得、アクティビティログ
- **ランキング**: 週次でのポイント競争

---

## 起動方法 (Local Network)

家庭内LANで他の端末（スマホなど）からアクセスするために、以下のコマンドでサーバーを起動します。

1. **データベースのセットアップ** (初回のみ)
   ```bash
   python3 manage.py makemigrations accounts chores gamification
   python3 manage.py migrate
   ```

2. **サーバー起動**
   ```bash
   # 0.0.0.0 を指定することで外部からのアクセスを許可します
   python3 manage.py runserver 0.0.0.0:8000
   ```

3. **アクセス方法**
   - **PC (サーバー稼働機)**: [http://localhost:8000](http://localhost:8000)
   - **スマホ等**: `http://<PCのIPアドレス>:8000`
     - MacのIPアドレス確認: `ipconfig getifaddr en0` (Wi-Fiの場合)

## 利用フロー

### 1. 最初のユーザー (世帯主)
1. **Sign Up**: アカウントを作成します。
2. **Setup**: 「新しいグループを作る」を選択し、家の名前（例：「田中家」）を入力します。
3. **Invite**: ダッシュボードに表示される「招待コード (8桁)」を同居人に伝えます。
4. **Register Chores**: 「家事タスク一覧へ」 -> 「+ 新しい家事」から、よくある家事を登録します。
   - 例: 「皿洗い (10pts, 普通)」, 「風呂掃除 (20pts, 少し大変)」

### 2. 同居人
1. **Sign Up**: 自分のスマホからアクセスし、アカウント作成します。
2. **Join**: 「招待コードで参加する」を選択し、教えてもらったコードを入力します。
3. **Dashboard**: 同じグループに参加でき、登録された家事が見えるようになります。

### 3. 家事の実行
1. 家事を片付けます。
2. スマホでアプリを開き、タスク一覧から該当する家事の「やった！」ボタンを押します。
3. ポイントが加算され、ダッシュボードの活動履歴に表示されます。

4. **Ranking & Analysis**:
   - 「ランキングを見る」で今週の順位を確認できます。
   - 「Analysis」タブから、過去30日間の「貢献度（円グラフ）」と「よくやる家事（棒グラフ）」を分析できます。
   - 家事を完了すると、画面に紙吹雪が舞う演出があります（要JavaScript）。
