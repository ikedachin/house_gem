# HouseGem (Housework Gamification App)

家族や同居人との家事分担を楽しくするWebアプリケーションです。
家事を「タスク」として管理し、完了するとポイントを獲得。週ごとにランキングを競うことができます。

※Google Antigravityで作成しました。作成時のプロンプトはcreate_prompt.txtに記載しています。人の手を介しているのはcreate_prompt.txtと仮想環境づくり、このREADME.mdです。

## ✨ 機能
- **グループ管理**: 招待コードで簡単に家族をグループに追加できます。
- **家事タスク**: よくやる家事を登録し、難易度に応じたポイントを設定。
- **実行記録**: 「やった！」ボタン一つで完了報告。
- **アクティビティ**: 誰がいつ何をやってくれたかが一目でわかります。
- **ランキング**: 週次で一番家事を頑張った人を称えましょう。

## 🚀 使い方 (Local Network)

このアプリは家庭内LAN（Wi-Fiなど）で動作することを想定しています。

### 1. 準備 (初回のみ)
```bash
# 仮想環境の作成と有効化 (uvを使用する場合)
uv venv
source .venv/bin/activate

# 依存パッケージのインストール
pip install -r requirements.txt

# データベースのセットアップ
python manage.py migrate
```

### 2. サーバー起動
PCのIPアドレスで公開するために `0.0.0.0` を指定して起動します。
```bash
python manage.py runserver 0.0.0.0:8000
```

### 3. アクセス方法
- **PC (サーバー稼働機)**: [http://localhost:8000](http://localhost:8000)
- **スマホ等**: `http://<PCのIPアドレス>:8000`
    - MacのIP確認: ターミナルで `ipconfig getifaddr en0` を実行

## 📱 利用ガイド

### 世帯主（最初のユーザー）
1. **新規登録**してアカウントを作成。
2. 「新しいグループを作る」で家の名前（例：「田中家」）を登録。
3. ダッシュボードに表示される **8桁の招待コード** を家族に共有。
4. 「家事タスク一覧」から、「皿洗い」「ゴミ出し」などを追加。

### ご家族・同居人
1. **新規登録**してアカウントを作成。
2. 「招待コードで参加」を選び、共有されたコードを入力。
3. これで同じ「家」のメンバーとして参加完了！
